# KeyDoor

An object the unlocks specific KeyDoors

## Structure
<pre>
▿ 📚 KeyDoor
    ▫️ 🔲 DoorPart  
    ▿ 🔲 Hitbox 
        ▫️ ⭕️ SelectionBox
</pre>

## When Disabled
<pre>
The KeyDoor cannot be unlocked
</pre>

## Attributes
<pre>
<b>KeyId</b>  
<hr>
▫️ <b>Type:</b> string  
▫️ <b>Description:</b> Keys with the same KeyId will unlock this KeyDoor
</pre>
