# SpecialEffectsRemover

A Part that will return the effects of the game back to normal

## Structure
<pre>
▿ 🔲 SpecialEffectsRemover
    ▫️ 🟪 TweenConfiguration
</pre>

## When Disabled
<pre>
Will not return the effects of the game back to normal
</pre>

<pre>
