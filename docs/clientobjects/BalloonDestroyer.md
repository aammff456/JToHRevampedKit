# BalloonDestroyer

A Part that can pop Balloons that touch it

## Structure
<pre>
▫️ 🔲 BalloonDestroyer <sup><b>Tag:</b> <i>ClientObject</i></sup>
</pre>

## When Disabled
<pre>
Will not be able to pop Balloons
</pre>
